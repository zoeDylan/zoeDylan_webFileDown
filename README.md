#网络文件下载#
>####给一个文件地址,然后自动下载文件！


##使用方式

    服务器地址?list=数组参数

###示例

    192.168.1.1:3000?list=[{name:"",url:"",fUrl:""},{name:"",url:"",fUrl:""}]
